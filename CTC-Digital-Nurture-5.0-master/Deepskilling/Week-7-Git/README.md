# Week 7 – Git Hands-On

## Overview

This repository contains the Week 7 Git Hands-On exercises completed as part of the **CTC Digital Nurture 5.0 – Deepskilling Program**. The exercises focus on Git fundamentals, repository management, branching, merging, conflict resolution, and working with remote repositories.

---

## Technologies Used

- Git
- GitHub
- Visual Studio Code
- PowerShell

---

## Exercises Completed

### Exercise 1 – Git Configuration

- Verified Git installation
- Configured Git username and email
- Initialized a Git repository
- Created and committed files
- Learned basic Git commands:
  - `git init`
  - `git status`
  - `git add`
  - `git commit`

---

### Exercise 2 – Git Ignore

- Created `.gitignore`
- Ignored `.log` files
- Ignored `log/` directory
- Verified ignored files using `git status`

---

### Exercise 3 – Branching and Merging

- Created a new branch
- Switched between branches
- Added and committed changes
- Merged branch into `master`
- Deleted merged branch

---

### Exercise 4 – Merge Conflict Resolution

- Created conflicting changes
- Resolved merge conflicts
- Completed merge successfully
- Updated `.gitignore`
- Cleaned up branches

---

### Exercise 5 – Remote Repository

- Verified repository status
- Listed branches
- Connected local repository with GitHub
- Pulled latest changes
- Pushed local repository to GitHub
